# Vicky 🧠

- init        > create vault + extract code graph
- query       > BFS graph + vault search
- remember    > save note to vault
- recall      > search vault notes
- extract     > refresh code graph
- help        > print this file

Graphify (bundled via `graphifyy` npm dep): `graphify --help`
Docs: https://github.com/rhanka/graphify

Obsidian [help](https://github.com/obsidianmd/obsidian-help/tree/master/en
): `obsidian --help`
